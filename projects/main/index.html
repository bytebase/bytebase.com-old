A Porsche without proper tires

“I've spent most of my career working on database. From the cross-platform GUI integrated database to the leading public cloud provider and then the top TPC-C performer.

While we have decent tools to manage database engine itself, our tooling around managing application-initiated db changes are a bit lacking.

What's the GitLab for database instead of for code? What's the Figma for DBAs instead of for designers?

There was no good answer up until Bytebase sets out to respond. ”

Tianzhou Chen
Creator of Bytebase
